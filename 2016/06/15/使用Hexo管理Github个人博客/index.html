    创建Github账号

    Tips

    • Github是非常好的团队合作平台，尤其是代码的存放。
    • 你也可以使用Github存放一些对自己很有用的文档。
    • 由于Github 免费版是公开的，你的所有Public仓库都能被别人看到或者搜索到。一些重要的信息不要放置在上面，比如 密码账号等

    1、 注册Github账号 （有Github账号的可以直接跳转到第二步，创建个人博客）

    1. 点击访问Github官网Github
    2. 点击Sign in进行注册
      注册过程比较简单，此处不再赘述，如果不会注册，建议你放弃接下来的教程，安安静静的享受人生。

    2、 创建个人博客

    1. 点击 + 选择 New repository

    2. 填写信息

      Warning

      • 仓库名称是 固定的 ： 名称+github.io 。 我的就是 SongFuZhen.github.io
      • 如果名称写错，将创建正常的仓库，而不是个人博客。之后将无法继续
      • 如果想创建Private， 需要购买Github服务，所以一般都是选择Public

      3、 测试使用

      看到如下界面就说明已经创建成功。 Congratulations!!! 请继续阅览

    安装Hexo

    如果你还没有安装Node.js和 Git Bash,那么请你先安装好再继续。如果安装了，可以使用下面方法测试

    1、参照Hexo官方文档安装 Git Bash 和 Node.js

    点击此处查看 如何安装Hexo环境

    2、测试

    1. 鼠标右击任意桌面，选择 Git Bash Here ， 打开如下图
    2. 测试 Node.js 是否成功， 在Bash里面输入 node之后 回车, 如下图

    3、安装Hexo环境

    如果全部安装成功，那么就可以安装Hexo环境了。

    1. 在Git Bash中输入一行命令 npm install -g hexo-cli
    2. 等待下载Hexo必备信息，时间视网速环境而定。可能时间稍微长点。出现下面信息表示安装成功
    3. 在Git Bash中输入 hexo -v， 出现下面信息表示安装成功

    4、创建Hexo目录

    Tips

    • 可以参照Hexo官网的建站, 点击查看如何使用Hexo建站
    • 在你要存放博客的文件夹中 右击， 打开 Git Bash。
    • 在命令行中输入 hexo init TestBlog。 意思就是初始化一个名为Blog文件夹。当然，名字你可以随便起
    • 等待，可能需要一会时间。 创建成功， 如下图
    • 进入新建目录， 输入命令行： cd TestBlog
    • 执行 npm install 命令， 输入命令行： npm install
    • 执行启动服务命令， 输入命令行： hexo s, 出现下面信息表示运行成功
    • 打开浏览器，输入url： localhost:4000

      If it show , Congratulations, Man! Just Enjoy Hexo。OOO
